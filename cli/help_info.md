# Ollama Wrapper

Usage:
* ai [prompt]
* ai /[command]

Available commands:
* /h - shows this message
* /i - shows info
---
* /c - cleans context
* /s - saves context
* /l - loads context
* /r - removes context
* /t - shows full conversation
---
* /u - starts container
* /d - stops container
* /o - manage ollama (run 'ai /o help' for more info)
* /m - change model
---

To add a system prompts ad it in system_prompts.json
